*main function*
The main function of ICoEF-CA is in file Test.m. During execution, the corresponding data set path needs to be changed, eg:path = 'D:\Files\Code\Mine\ICoEF-CA-matlab\Dataset\'.

*ICoEF-CA algorithm*
The ICoEF-CA algorithm is in file main.m.

*data set*
The data set is in the file/Dataset eg:Dataset/BarabasiAlbert_n2500m1.mat. This data set has been converted to adjacency matrices. 

*result*
Result stores the run results, including HV1,ParetoFront,T.
